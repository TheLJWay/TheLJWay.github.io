Suivre

Suivre (French for "to follow") is an Android budget managning application. As college students, my team aimed to make an easy to use app that we could use ourselves to follow our spending habits. My contributions for this role was creating the initial mockups and design that we wanted to achieve for our end product. We were very ambitious with this project, but the time contraints we had learning Android Studio for the first time and prioritizing on getting main features down did not give us the time to make the final product as presentable as we initially hoped. This project was a learning curve for everyone involved, but for me it was worthwhile.
